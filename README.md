# flag-face-filter

A simple computer vision project that detects faces using OpenCV and applies a transparent Indian tricolor overlay with the Ashoka Chakra on the face in real time through a webcam.

# Features
- Real-time face detection
- Tricolor face overlay
- Ashoka Chakra design
- Webcam based filter
- Independence Day greeting text

# Technologies
- Python
- OpenCV
- NumPy

# Installation

Install required libraries:

opencv-python
numpy
Run
python main.py

Press Q to exit the application.
